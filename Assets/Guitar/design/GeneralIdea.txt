피해량 공식

(가하는 피해량) * 80 / (80 + (방어력) * (1 - 방어 관통)) * (피해량 배율)

80 / (80 + (방어력) * (1 - 방어 관통)) => 기본 피해 감소율

게임 내부에서는 int 타입을 사용하기 때문에 
8000 / (8000 + (방어력) * (1 - 방어 관통))
으로 판정한다, 알겠지?

게임 개시 시 맵 온갖 군데에 버프 아이템 생성 + 일정 시간 뒤에 거점 생성

버프 아이템은 거점 장악 효율 증가, 거점에 두 팀이 다 들어가있을 경우 장악이 막히는 대신 서로가 장악을 개시

먼저 거점을 장악한 팀이 승리하기 때문에 거점 장악 효율 아이템을 많이 모을수록 유리

장악형(브루저/탱커)은 뛰어난 생명력과 높은 지속 전투력으로 거점 장악에 실제 뛰어남

기동형(암살자/교란자)은 빨라서 거점 장악 효율을 올리는 버프 아이템을 모으기 쉬우나 거점 안에서는 기동력을 다소 상실하기에 장악 자체는 못하는 편
-> 거점을 장악하기 위해서는 거점 안에 계속 있어야 하나, 기동력을 발휘하려면 거점 밖으로 나가게 됨

압박형(레인저/스나이퍼)은 거점 장악 중인 장악형에게 압박을 넣어서 빼게 만들기 좋지만 근접 전투에 불리

조력형(힐러/인보커(가칭))은 나머지 6 클래스보다 기본 스펙이 딸리지만 나름의 전투력에 더해 아군 지원 효과가(힐러: 지속 전투와 생존, 인보커: 공격력 강화, 추격과 처치) 탁월

예상되는 일반 조합은 장기압 / 장기지 / 장압지
